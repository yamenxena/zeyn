# Graph UI Heuristics (Weaver Agent)

## 1. Pure Visual Architecture
The UI is a naive renderer that perfectly visualizes dynamic JSON trees.
- It MUST NOT possess Agent-control mechanisms like "Run Seeker Agent" buttons.
- The UI MUST be deeply appealing, cohesive, and use uniform character portraits processed into glassmorphic components.

## 2. Character Portraits (The Hero Circles)
- Instead of raw text, the focal point of a Node interaction is a unified Marvel portrait.
- Images generated dynamically or mapped must preserve original character colors while harmonizing with the dark space theme.

## 3. The OneZoom Fractal Paradigm
- Render macroscopic JSON aggregations offloading node calculations.
- Maintain top-aligned ribbon sorting (Chronology vs. Power) without triggering database refreshes.
